# 🌍 Global LNG Production & Contract Modelling System  - A Python Case Study

## 📌 Overview

This repository shares my learnings in building and designing a modelling system to represent the global LNG market, in particular global LNG production and long-term contract flows at a:

- Geographical level (Region/Country)   
- Liquefaction plant & project level  
- Shareholder level  

The system was designed to support forward-looking commercial analysis, supply forecasting, and scenario modelling.
